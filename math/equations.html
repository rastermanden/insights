How it works

m (slope) — how steep the line is. For every 1 step right, y changes by m. Positive = goes up, negative = goes down, zero = flat.

b (y-intercept) — where the line crosses the y-axis (x = 0).

Root (x-intercept) — where y = 0, solved as x = −b / m.

How it works

a — controls width and direction. a > 0 opens upward (U-shape, minimum). a < 0 opens downward (∩-shape, maximum). Larger |a| = narrower.

b — shifts the vertex left or right. Vertex x = −b / (2a).

c — the y-intercept (value at x = 0).

Roots — where the parabola crosses y = 0, found via the quadratic formula: x = (−b ± √(b²−4ac)) / 2a. The discriminant b²−4ac tells you how many real roots exist.

How the solver works

Linear (ax + b = c) — rearrange to ax = c − b, then x = (c − b) / a. Requires a ≠ 0.

Quadratic (ax² + bx + c = 0) — uses the quadratic formula: x = (−b ± √(b²−4ac)) / 2a.

The discriminant b²−4ac determines the number of solutions: positive → two real roots, zero → one repeated root, negative → no real roots (complex).
