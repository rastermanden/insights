How it works

Linear regression finds the line y = mx + b that minimises the sum of squared vertical distances from each point to the line.

Slope (m) — how much y changes for each unit increase in x.

Intercept (b) — the value of y when x = 0.

R² — goodness of fit (0–1). A value of 1 means all points lie exactly on the line; 0 means the line explains none of the variation.
