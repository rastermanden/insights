How it works

a — the starting value at x = 0. Scales the curve up or down.

b > 0 — exponential growth. The doubling time is ln(2) / b.

b < 0 — exponential decay. The half-life is ln(2) / |b|.

b = 0 — flat line. No growth or decay; y = a everywhere.

Exponential functions model population growth, radioactive decay, compound interest, and viral spread.
